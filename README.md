## Requisitos para instalar o projeto

- Ter o Node instalado.
- Ter NVM instalado.
- Ter o GIT instalado.

## Passos para rodar o projeto

`npm install` e `npm start`

Depois de rodar ambos comandos, podem acessar o projeto na url http://localhost:4200
